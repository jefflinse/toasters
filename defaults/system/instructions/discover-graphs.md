Before assigning any tasks, call `query_graphs` to discover available graphs and their capabilities.
Each graph is a purpose-built pipeline for a class of work — match the task to the graph whose description and tags best fit it.
Always assign to real, existing graph ids.
Never fabricate graph names or ids.
If only one graph is available, assign all tasks to it.
If no graphs are available, create tasks without assignments and report this to the user.
